# 영상 편집 기능 요구사항

## 영상 편집 기능의 중요성

이 플랫폼의 가장 중요한 기능은 경기 영상 편집이다.

스케줄, 공지사항, 선수 기록도 필요하지만, 실제 판매 관점에서 가장 중요한 경쟁력은 지도자가 경기 영상을 얼마나 쉽고 유용하게 분석할 수 있는지이다.

따라서 영상 편집 기능은 단순히 영상을 자르는 기능이 아니라, 지도자가 선수에게 전술적 의도를 전달할 수 있는 분석 도구로 설계해야 한다.

## 기본 영상 기능

### 영상 재생

업로드된 경기 영상을 재생할 수 있어야 한다.

필요한 기능은 다음과 같다.

* 재생
* 일시정지
* 앞으로 이동
* 뒤로 이동
* 전체화면
* 현재 재생 시간 확인
* 전체 영상 길이 확인

### 재생 속도 조절

지도자가 중요한 장면을 천천히 확인할 수 있어야 한다.

지원할 재생 속도는 다음과 같다.

* 0.25x
* 0.5x
* 1x

### 프레임 단위 이동

공 터치 순간, 오프사이드 라인, 슈팅 직전 장면처럼 세밀한 분석이 필요한 경우 프레임 단위로 이동할 수 있어야 한다.

프레임 이동은 영상 분석 퀄리티에 직접적인 영향을 주는 기능이다.

## 클립 기능

### 구간 자르기

지도자는 전체 경기 영상 중 필요한 구간만 선택해서 클립으로 저장할 수 있어야 한다.

예시:

* 전반 12:30 ~ 13:10
* 후반 65:20 ~ 66:05

클립 저장 시 필요한 정보는 다음과 같다.

* 원본 경기 영상
* 시작 시간
* 종료 시간
* 클립 제목
* 클립 메모
* 클립 유형
* 팀 분석용 또는 선수 개인 분석용 여부

### 클립 제목/메모

클립에는 분석 목적을 알 수 있도록 제목과 메모를 작성할 수 있어야 한다.

예시:

* 후방 빌드업 실수
* 수비 전환 늦음
* 압박 타이밍 좋음
* 전진 패스 가능 장면
* 세트피스 수비 위치 문제

### 클립 묶음

클립은 분석 목적에 따라 묶어서 관리할 수 있어야 한다.

예시:

* 공격 장면
* 수비 장면
* 전환 장면
* 세트피스 장면
* 개인 피드백 장면

## 캡처 기능

지도자는 원하는 시간대의 장면을 이미지로 저장할 수 있어야 한다.

캡처 이미지는 다음 용도로 사용할 수 있다.

* 전술 설명
* 선수 피드백

## 북마크 기능

지도자는 중요한 장면에 북마크를 남길 수 있어야 한다.

북마크는 전체 영상을 보다가 나중에 다시 확인해야 하는 장면을 표시하는 용도로 사용한다.

북마크에는 다음 정보가 포함될 수 있다.

* 영상 ID
* 북마크 시간
* 제목
* 메모
* 작성자

## 드로잉 기능

드로잉 기능은 지도자가 영상 위에 직접 전술적인 설명을 남기기 위한 기능이다.

### 선 그리기

수비 라인, 패스 라인, 이동 라인을 표시할 수 있어야 한다.

### 화살표

선수 이동 방향, 패스 방향, 압박 방향을 표시할 수 있어야 한다.

### 원/박스 표시

특정 선수나 특정 공간을 강조할 수 있어야 한다.

### 영역 표시

압박 구역, 하프스페이스, 위험 지역 등을 표시할 수 있어야 한다.

### 텍스트 메모

영상 위에 짧은 설명을 직접 작성할 수 있어야 한다.

예시:

* 여기서 전진 패스 가능
* 수비 라인 유지
* 반대 전환 필요
* 압박 시작 타이밍

### 확대/줌

특정 장면이나 특정 선수를 확대해서 볼 수 있어야 한다.

### 화면 정지 후 설명

영상을 멈춘 상태에서 전술판처럼 설명할 수 있어야 한다.

이 기능은 지도자가 선수에게 장면을 설명할 때 매우 중요하다.

### 드로잉 유지 시간 설정

드로잉은 특정 시간 동안만 표시될 수 있어야 한다.

예시:

* 5초 동안 화살표 표시
* 3초 동안 선수 강조
* 10초 동안 압박 구역 표시

## 코멘트 기능

지도자는 클립 또는 특정 시간대에 코멘트를 작성할 수 있어야 한다.

코멘트는 선수에게 분석 의도를 전달하기 위한 기능이다.

필요한 코멘트 유형은 다음과 같다.

* 클립 전체 코멘트
* 특정 시간대 코멘트
* 선수 개인 피드백 코멘트
* 팀 전체 피드백 코멘트

## 팀 분석 영상

팀 분석 영상은 팀 전체에게 공유되는 분석 영상이다.

사용 대상:

* 지도자
* 팀 소속 선수 전체

예시:

* 팀 압박 구조 분석
* 후방 빌드업 패턴 분석
* 세트피스 수비 분석
* 수비 전환 문제 분석

## 선수 개인 분석 영상

선수 개인 분석 영상은 특정 선수에게만 공유되는 분석 영상이다.

사용 대상:

* 지도자
* 해당 선수 본인

예시:

* 특정 선수의 수비 위치 문제
* 특정 선수의 패스 선택
* 특정 선수의 움직임
* 특정 선수의 압박 타이밍

## MVP 기준

초기 버전에서 반드시 필요한 기능은 다음과 같다.

1. 경기 영상 업로드
2. 영상 재생
3. 재생 속도 조절
4. 구간 선택
5. 클립 저장
6. 클립 제목/메모 작성
7. 팀 분석/개인 분석 구분
8. 선수 개인 영상 접근 제어
9. 기본 드로잉 도구
10. 코멘트 작성

## 추후 확장 기능

초기 버전에서 무리하게 구현하지 않고, 추후 확장할 기능은 다음과 같다.

* 자동 선수 추적
* 자동 이벤트 분석
* AI 패스 인식
* AI 슈팅 인식
* AI 압박 인식
* 자동 하이라이트 생성
* 선수별 움직임 분석
* 이벤트 기반 클립 자동 생성
