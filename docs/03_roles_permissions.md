# 03. 역할 및 권한 정책

## 1. 역할 구분

```text
COACH
ANALYST
PLAYER
```

현재 서비스는 단일 팀 기준이다.

권한 검증은 반드시 백엔드에서 수행한다.

---

## 2. COACH 권한

### 팀 운영

- 스케줄 등록·조회·수정·삭제
- 공지사항 등록·조회·수정·삭제
- 회원과 선수 관리

### 영상

- 경기 원본 영상 등록·조회·수정·삭제·복구
- 팀 분석 클립 등록·조회·수정·삭제
- 선수 개인 분석 클립 등록·조회·수정·삭제
- 드로잉·텍스트·정지·줌
- DRAFT와 스냅샷 관리
- 개별·일괄 렌더링
- FAILED 재시도
- 편집 잠금 강제 해제
- 삭제 파일 정리 상태 확인

### 선수 기록

- 경기별 선수 기록 등록·갱신
- 분석 클립 연결 이벤트 등록
- 전체 기록과 이벤트 조회

### 북마크

- 본인 북마크 CRUD

---

## 3. ANALYST 권한

### 팀 운영

- 스케줄 조회
- 공지사항 조회

팀 운영 등록·수정·삭제 권한은 기본적으로 없다.

### 영상

영상 도메인에서는 COACH와 동일한 전체 권한을 가진다.

- 경기 원본 영상 등록·조회·수정·삭제·복구
- 팀 분석 클립 등록·조회·수정·삭제
- 선수 개인 분석 클립 등록·조회·수정·삭제
- 드로잉·텍스트·정지·줌
- DRAFT와 스냅샷 관리
- 개별·일괄 렌더링
- FAILED 재시도
- 편집 잠금 강제 해제

기존 “ANALYST 영상 삭제 불가” 정책은 폐기한다.

### 선수 기록

- 경기별 선수 기록 등록·갱신
- 분석 클립 연결 이벤트 등록
- 전체 기록과 이벤트 조회

### 북마크

- 본인 북마크 CRUD

---

## 4. PLAYER 권한

### 팀 정보

- 스케줄 조회
- 공지사항 조회

### 영상

- 접근 가능한 경기 원본 영상 조회
- READY 팀 분석 클립 조회
- 본인의 READY 선수 개인 분석 클립 조회
- 본인 개인 클립 조회 기록 생성·갱신

다음 상태에는 접근할 수 없다.

```text
DRAFT
QUEUED
PROCESSING
FAILED
```

### 선수 기록

- 본인 기록 조회
- 본인 이벤트 조회

### 금지

- 영상 등록·수정·삭제·복구
- 편집기 접근
- 렌더링 요청
- 다른 선수 개인 클립 조회
- 북마크 API
- 선수 기록 쓰기 API

---

## 5. 영상 접근 정책

### 경기 원본 영상

로그인 사용자에게 허용된 공통 조회 범위에서 제공한다.

삭제 상태이거나 실제 파일이 없으면 재생할 수 없다.

### 팀 분석 클립

- COACH
- ANALYST
- 팀 소속 PLAYER

PLAYER에게는 READY만 제공한다.

### 선수 개인 분석 클립

- COACH
- ANALYST
- 대상 선수 본인

같은 팀의 다른 PLAYER는 접근할 수 없다.

### 편집 데이터

DRAFT, 스냅샷, 잠금 토큰과 편집 원본 데이터는 COACH·ANALYST만 접근한다.

---

## 6. 삭제 권한

### 원본 경기 영상

COACH와 ANALYST가 삭제·복구한다.

- 10일 복구 가능
- QUEUED·PROCESSING 연결 클립이 있으면 삭제 차단

### 분석 클립

COACH와 ANALYST가 DRAFT, FAILED, READY를 영구 삭제한다.

QUEUED와 PROCESSING은 삭제할 수 없다.

### 북마크

COACH와 ANALYST가 본인 북마크만 삭제한다.

---

## 7. 권한 검증 기준

- JWT 인증 사용자 확인
- `memberRole` 확인
- PLAYER 본인 여부 확인
- 북마크 작성자 확인
- 개인 클립 대상 선수 확인
- 삭제 상태와 클립 상태 확인
- 파일 존재 여부 확인
- 프론트 버튼 노출과 무관하게 Service 또는 Validator에서 재검증
- `isAdmin`만으로 영상 권한 부여 금지

---

## 8. API 주소 방향

```text
공통 조회: /api/{resources}
관리용: /api/management/{resources}
선수 본인: /api/player/me/{resources}
관리자: /api/admin/{resources}
```

기존 `/api/coach/**` 영상 삭제 API는 영상 도메인 권한 변경에 맞춰 소스 검토 후 관리용 주소로 이동하거나 임시 호환한다.

새 주소를 실제 Controller 확인 없이 임의로 확정하지 않는다.
