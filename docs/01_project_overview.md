# 프로젝트 개요

## 프로젝트 목적

이 프로젝트는 실제 축구팀에 운영할 목적으로 개발하는 플랫폼이다.

단순히 기능을 구현하는 것이 목적이 아니라, 지도자와 선수가 실제 팀 운영과 경기 분석에 사용할 수 있는 서비스를 만드는 것이 목적이다.

## 플랫폼 개요

축구팀에서 사용하는 팀 운영 및 경기 영상분석 플랫폼이다.

지도자는 팀 일정, 공지사항, 경기 영상, 분석 클립, 선수 개인 피드백 영상을 관리한다.

선수는 모바일에서 스케줄과 공지사항을 확인하고, 팀 분석 영상과 본인 개인 분석 영상을 시청한다.

## 주요 사용자

### 지도자

지도자는 팀을 운영하고 분석 영상을 만드는 사용자이다.

주요 역할은 다음과 같다.

* 스케줄 관리
* 공지사항 관리
* 경기 영상 업로드
* 경기 영상 편집
* 팀 분석 영상 생성
* 선수 개인 분석 영상 생성
* 선수 기록 확인

### 선수

선수는 지도자가 등록한 정보를 확인하고 분석 영상을 시청하는 사용자이다.

주요 역할은 다음과 같다.

* 스케줄 확인
* 공지사항 확인
* 팀 분석 영상 시청
* 본인 개인 분석 영상 시청
* 본인 기록 확인

## 핵심 기능

* 팀 관리

* 공지사항 관리
* 경기 영상 업로드

* 선수별 기록 확인

## 가장 중요한 기능

이 플랫폼의 가장 중요한 기능은 경기 영상 편집이다.

스케줄이나 공지사항도 필요하지만, 플랫폼의 경쟁력은 지도자가 얼마나 쉽고 정확하게 경기 영상을 분석할 수 있는지에 달려 있다.

따라서 영상 편집 기능은 처음부터 가장 중요한 기능으로 보고 설계한다.

## 서비스 사용 환경

선수는 대부분 모바일에서 서비스를 사용할 가능성이 높다.

지도자는 경기 영상을 업로드하고 편집해야 하기 때문에 PC 사용 비중이 높다.

따라서 화면은 반응형으로 개발하되, 선수 화면은 모바일 사용성을 우선하고 지도자 편집 화면은 PC 사용성을 우선한다.

## 개발 우선순위

1. 회원, 팀, 권한 구조
2. 경기 영상 업로드 구조
3. 영상 편집 MVP
4. 팀 분석 영상 공유
5. 선수 개인 영상 접근 제어
6. 스케줄 관리
7. 공지사항 관리
8. 선수 기록 관리
9. AI 자동 분석 기능 확장
