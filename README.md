# Soccer Team Management Platform

## 프로젝트 개요

이 프로젝트는 축구팀에서 사용할 수 있도록 개발하는 축구팀 운영 및 경기 영상분석 플랫폼이다.

지도자는 팀 일정, 공지사항, 경기 영상 업로드, 경기 영상 편집, 팀 분석 영상, 선수 개인 분석 영상을 관리할 수 있다.

선수는 모바일 환경에서 스케줄과 공지사항을 확인하고, 팀 분석 영상과 본인 개인 분석 영상을 시청할 수 있다.

이 플랫폼의 가장 중요한 핵심은 단순한 영상 업로드가 아니라, 지도자가 경기 영상을 쉽게 분석하고 선수에게 명확하게 전달할 수 있는 영상 편집 기능이다.

## 주요 사용자

* 지도자
* 선수

## 주요 기능

### 지도자 기능

* 스케줄 등록, 수정, 삭제
* 공지사항 등록, 수정, 삭제
* 경기 영상 업로드
* 경기 영상 편집
* 팀 분석 영상 생성
* 선수 개인 분석 영상 생성
* 모든 분석 영상 조회
* 선수별 기록 확인

### 선수 기능

* 스케줄 조회
* 공지사항 조회
* 팀 분석 영상 조회
* 본인 개인 분석 영상 조회
* 본인 기록 확인

## 영상 분석 기능

영상 편집 기능은 이 플랫폼의 핵심 기능이다.

지도자는 경기 영상에서 중요한 장면을 클립으로 저장하고, 해당 장면에 선, 화살표, 원, 박스, 텍스트, 코멘트 등을 추가할 수 있어야 한다.

예를 들어 다음과 같은 분석이 가능해야 한다.

* 후방 빌드업 실수 장면 클립화
* 수비 전환이 늦은 장면 표시
* 패스 라인 표시
* 수비 라인 표시
* 특정 선수 강조
* 위험 지역 표시
* 클립별 코멘트 작성
* 선수 개인 피드백 영상 생성

## 개발 방향

초기 버전에서는 AI 자동 분석보다 지도자가 직접 사용할 수 있는 수동 영상 편집 기능의 완성도를 우선한다.

이후 확장 기능으로 자동 선수 추적, 자동 이벤트 분석, AI 패스/슈팅/압박 인식 기능을 고려한다.

## 핵심 목표

이 프로젝트의 중심은 단순한 CRUD 기능이 아니다.

가장 중요한 목표는 지도자가 경기 영상을 쉽게 편집하고, 분석 내용을 선수에게 정확하게 전달할 수 있는 플랫폼을 만드는 것이다.
